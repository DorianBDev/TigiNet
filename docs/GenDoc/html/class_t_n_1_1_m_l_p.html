Detailed Description

Multi Layer Percreptron class.

Use a simple multi layer perceptron with all included.

TN::MLP::MLP ( double  errorAcceptance = 0.1,
double  learningLevel = 0.01,
double  momentum = 0.0001,
double(*)(double)  activationFunction = &Sigmoide,
size_t  layerCount = 2,
unsigned int *  neuronePerLayerTable = NULL 
)

Create a MLP network.

Parameters
errorAcceptance: error step of the MLP (between 0 and 1).
learningLevel: learning level of the MLP (between 0 and 1).
momentum: the momentum of the newtork, to get through local minimum (between 0 and 1).
activationFunction: the activation function (see available functions in Math class)
layerCount: the number of layers in the network.
neuronePerLayerTable: a table with all the neurones counts per layer.

void TN::MLP::Learn ( double **  valIn,
double **  valOut,
size_t  size 
)

Learn from matrices with size training elements.

Each training elements for the input matrix need a size of the neurone count in the first layer. Same for output but with the last layer neurone count. e.g. valIn[size][neuroneCountInFirstLayer]

Parameters
valIn: the input matrix with size line and neurone count in first layer row.
valOut: the output matrix with size line and neurone count in last layer row.
size: the number of learning samples.
